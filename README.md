# 🎬 OpenClaw Tools

OpenClaw AI 视频处理工具集 | Made with ❤️ by LeoKan

## 功能

### 1. 视频素材下载
- 支持 Pexels 免费视频素材
- 自动搜索关键词
- 批量下载

### 2. 视频拼接
- 使用 MoviePy 自动拼接
- 统一分辨率
- 支持转场效果

### 3. 后续功能开发中
- AI配音
- 自动字幕
- 文案生成

## 使用方法

`ash
# 下载视频素材
python scripts/auto_gen.py

# 拼接视频
python scripts/concat_story.py
`
